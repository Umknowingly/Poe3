# Poe3
BIMO is a Windows desktop app that helps you learn about cybersecurity, manage security tasks, and test your knowledge through quizzes.

Features
Chat Assistant - Ask about passwords, phishing, scams, malware, 2FA, VPN, and more

Task Manager - Add, complete, and delete security tasks with MySQL database storage

Interactive Quiz - Test your cybersecurity knowledge with 12+ questions

Activity Log - Track all your actions and interactions

Details
Frontend: WPF (Windows Presentation Foundation)

Language: C# (.NET Framework 4.8)

Database: MySQL Server

Connector: MySql.Data

Requirements
Windows 7 SP1 or later

.NET Framework 4.8

MySQL Server 5.7+
